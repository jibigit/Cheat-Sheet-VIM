The Cheat Sheet VIM

Les commandes de bases de Vim

Pour ouvrir Vim
Taper vim et entrée dans le terminal

Se déplacer vers le haut
K

Se déplacer vers le bas
J

Se déplacer vers la gauche
H

Se déplacer vers la droite
L

Pour quitter Vim sans enregistrer
:q

Pour quitter Vim en enregistrant
Taper Echap :wq

Pour effacer un caractère sous le curseur en mode normal
tapez xtq

Pour insérer du texte au niveau du curseur en mode normal. pouvez quitter en tapant
<Échap>
tapez i et insérez votre texte

Naviguer entre les modes

Le mode Normal.
Quelque soit votre mode actuel, vous pouvez revenir au mode normal en appuyant sur
Echap

Pour activer le mode Insertion depuis le mode normal, il faut taper
i

Pour activer le mode Remplacement depuis le mode normal, il faut taper
R

Vous pouvez aussi remplacer uniquement le caractère sous le curseur avec
r

Vous tapez ensuite le nouveau caractère, et l'éditeur repasse automatiquement en mode normal.

Copier/Couper-Coller des lignes entières

Supprime les lignes et les mets dans le presse papier
dd


Pour les coller, il suffit, en mode normal, d'appuyer sur la touches
p

Supprime les lignes et les mets dans le presse papier
p

Pour copier, la commande est
yy

Pour coller, la commande est toujours
p

Pour annuler

Toujours en mode normal, taper:
u

Aller à la ligne x

En mode normal, vous pouvez envoyer le curseur à la ligne 18 en tapant 18

Vous pouvez aussi aller en haut du fichier en tapant gg

ou en bas en tapant g

Rechercher du texte

En mode normal, pour rechercher la chaine "foobar", tapez
/foobar

Copier/Couper-Coller en mode visuel

Le mode visuel vous permet de sélectionner du texte. Sélectionnez le texte que vous voulez couper ou copier, et appuyez sur
d pour couper
y pour copier
p pour coller
